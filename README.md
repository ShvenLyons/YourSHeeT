# YourSheeT

一个用于记录时间段密文的网页工具。

<p>
  <a href="https://shvenlyons.github.io/YourSHeeT/">
    <strong>打开 YourSheeT</strong>
  </a>
</p>

## 使用方法

1. 打开上方链接进入网页。
2. 在 `Begin` 中输入开始时间。
3. 在 `End` 中输入结束时间；如果没有后续，勾选 `End 为 N/A`。
4. 页面会自动生成密文。
5. 需要解析密文时，在解密区域按页面提示填写内容。
6. 需要保存记录时，在记录区域输入日期和密文列表，然后点击保存。
7. 点击 `查看记录` 可以进入记录页。

## 记录页

记录页会显示当前仓库中的记录，并合并浏览器本地保存的记录。
